
Description:
	This project is called Crazy Tanks. It is a small single and multiple player game that allows players to compete with friends by destroying one another's tank.  


How to run:
	User should run the file "MainGame.py" in editor 

Libraries:
	This game uses Pygame module. 

Shortcut Commands:
	"r" to restart current game, the game will provide a different map, and location for tanks, but the scores will be the same.
	"2" to change the winning rounds to 2 instead of 10 rounds
	"p" to pause the game 

In Easter Mode:
	"1" to add 1000 points 
	"i" to be invinsible (will not be destroyed by bullets)
